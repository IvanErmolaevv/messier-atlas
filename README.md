# Messier Atlas

Статический интерактивный тренажёр для изучения объектов Мессье. Никакая сборка и сервер не требуются: сайт запускается из одного `index.html` и готов к GitHub Pages.

## Что реализовано

- Современная адаптивная карта неба со слоями созвездий, названиями ярких звёзд и объектами Мессье.
- Режим обучения: видны подписи и карточка с маршрутом поиска.
- Режим тренировки: выбрать объект, поставить метку на карте, проверить точность и увидеть верные позиции.
- Локальное сохранение лучшего результата в браузере.
- Каталог ключевых объектов текущего участка карты.

## Публикация в GitHub Pages

1. Создайте на GitHub новый репозиторий, например `messier-atlas`.
2. Загрузите **содержимое** этой папки в корень репозитория.
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main` и папку `/(root)`.
5. Сохраните. Через несколько минут сайт появится по адресу `https://ВАШ-ЛОГИН.github.io/messier-atlas/`.

Для локального просмотра достаточно открыть `index.html` в браузере.
