# Messier Atlas

Светлый интерактивный атлас и тренажёр объектов Мессье. Никакая сборка и сервер не требуются: сайт готов к GitHub Pages.

## Что реализовано

- Отдельная стартовая страница и вход через «Наблюдательный тур».
- Круговая карта настоящего неба: звёзды стоят на реальных координатах J2000, а не на декоративных точках.
- Корректные общепринятые европейские линии созвездий и русские популярные названия ярких звёзд.
- Каталог: вращаемая мышью или пальцем небесная полусфера 180°.
- Обучение: локальные реальные поля 36–40°; на каждой карте показано только несколько близких объектов Мессье.
- Тренировка: компактное поле 36° без подписей; новый объект и участок можно получать бесконечно.

## Астрономические данные

При открытии страница загружает открытые JSON-каталоги проекта [d3-celestial](https://github.com/ofrohn/d3-celestial), лицензия BSD-3-Clause:

- звёзды до 6-й звёздной величины: XHIP / Hipparcos;
- объекты Мессье;
- распространённые западные (европейские) фигуры созвездий;
- русские собственные имена звёзд.

IAU определяет стандартные названия, области и границы 88 созвездий; единственный обязательный рисунок «палочных» линий IAU не задаёт. Поэтому в атласе линии взяты из признанной западной схемы, а положения всегда вычисляются из реальных небесных координат.

## Публикация в GitHub Pages

1. Создайте на GitHub новый репозиторий, например `messier-atlas`.
2. Загрузите **содержимое** этой папки в корень репозитория.
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main` и папку `/(root)`.
5. Сохраните. Через несколько минут сайт появится по адресу `https://ВАШ-ЛОГИН.github.io/messier-atlas/`.

Для локального просмотра достаточно открыть `index.html` в браузере.
