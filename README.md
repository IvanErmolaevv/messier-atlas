# Messier & Stars Atlas

Светлый интерактивный атлас и тренажёр объектов Мессье и ярких звёзд. Никакая сборка и сервер не требуются: сайт готов к GitHub Pages.

## Что реализовано

- Отдельная стартовая страница и вход через «Наблюдательный тур».
- Круговая карта настоящего неба: звёзды стоят на реальных координатах J2000, а не на декоративных точках.
- Линии и астрономические имена из культуры неба **Modern / Современная** — стандартной культуры Stellarium.
- Каталог: отдельная вращаемая мышью или пальцем небесная полусфера 180° с отмеченными объектами Мессье.
- Обучение: сначала изучение локального поля 36–40°, затем обязательная расстановка всех меток. Участок засчитывается только после правильного размещения каждого объекта.
- Тренировка: вращаемая полусфера 180° без подписей объектов Мессье; метку нужно перетащить с панели на карту и отдельно проверить результат.
- Новый маршрут **«Звёзды»**: 200 ярких звёзд с собственными именами, сгруппированных в реальные локальные поля 30–40°. Сначала отображаются ответы, затем собственные имена скрываются и все звёздные метки нужно расставить точно.

## Астрономические данные

При открытии страница загружает открытые JSON-каталоги проекта [d3-celestial](https://github.com/ofrohn/d3-celestial), лицензия BSD-3-Clause:

- звёзды до 6-й звёздной величины: XHIP / Hipparcos;
- объекты Мессье;
- русские собственные имена звёзд.

Для фигур созвездий и первичного состава курса используется официальный файл культуры [Modern в Stellarium](https://github.com/Stellarium/stellarium/tree/master/skycultures/modern). Его линии заданы HIP-идентификаторами и напрямую сопоставляются с XHIP / Hipparcos, поэтому их положение на карте остаётся реальным. Русские подписи берутся из d3-celestial, при отсутствии перевода выводится имя из Stellarium.

Под «топ-200» здесь понимаются 200 разных ярчайших звёзд с собственными именами: набор сортируется по видимой звёздной величине, без повторов имён. На текущем каталоге это диапазон примерно от −1,44m до 3,4m.

## Публикация в GitHub Pages

1. Создайте на GitHub новый репозиторий, например `messier-atlas`.
2. Загрузите **содержимое** этой папки в корень репозитория.
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main` и папку `/(root)`.
5. Сохраните. Через несколько минут сайт появится по адресу `https://ВАШ-ЛОГИН.github.io/messier-atlas/`.

Для локального просмотра достаточно открыть `index.html` в браузере.
