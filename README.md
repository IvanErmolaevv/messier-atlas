# Messier Atlas

Светлый интерактивный атлас и тренажёр объектов Мессье. Никакая сборка и сервер не требуются: сайт запускается из одного `index.html` и готов к GitHub Pages.

## Что реализовано

- Отдельная стартовая страница и вход через «Наблюдательный тур».
- Светлые карты неба с фигурами созвездий, яркими звёздами и координатной сеткой.
- Обучение: каталоговый атлас со случайной бесконечной последовательностью участков.
- Тренировка: найти объект на карте без подписей; новый объект и участок можно получать бесконечно.

## Публикация в GitHub Pages

1. Создайте на GitHub новый репозиторий, например `messier-atlas`.
2. Загрузите **содержимое** этой папки в корень репозитория.
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main` и папку `/(root)`.
5. Сохраните. Через несколько минут сайт появится по адресу `https://ВАШ-ЛОГИН.github.io/messier-atlas/`.

Для локального просмотра достаточно открыть `index.html` в браузере.
